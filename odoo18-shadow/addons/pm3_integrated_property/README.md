# PM3 Integrated Property

Odoo 18 integrated property governance module suite.

This directory is the Odoo 18 implementation root for the PM3 Integrated Property design.

## Positioning

Odoo is the governance core. Google Workspace is the document and forum collaboration layer. LINE Official Account is the real-time interaction entry. Three-piece edge devices provide field runtime and service capacity. M5 metric policy governs permission, AI action limits, and auditability. Five-dimensional code is the identity credential and service entry.

## First implementation batch

- pm3_base: shared settings, sequences, base menus
- pm3_identity: five-dimensional identity, credentials, identity requests
- pm3_community: communities and households
- pm3_governance: organizations, roles, assignments
- pm3_policy: M5 policy rules and decision logs
- pm3_audit: global access audit log

## Safety boundary

This repository change only creates module files. It does not install modules, upgrade databases, restart containers, or modify live Odoo data.
